# geetest

* 极验验证码-缺口验证码破解
* 支持版本: ![](https://img.shields.io/badge/Python-3.x-blue.svg)

### 下载安装

* 下载源码:

```
https://github.com/pythonyhd/geetest
```

* 安装依赖:

```shell
pip install -r requirements.txt
```

### 使用方法

    * 首先运行captch_handles生成处理后的验证码
    * 执行main文件测试验证码识别效果，简单的w3c注册滑块破解
